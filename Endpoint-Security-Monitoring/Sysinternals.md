# Sysinternals

## Objective
This room covers the Sysinternals suite of tools used to analyze Windows systems and applications in depth, going beyond what's available in Task Manager. These tools are commonly used by SOC analysts and incident responders to investigate running processes, network connections, and system behavior during live investigations.

## Skills Demonstrated
- Using Sysinternals tools to analyze Windows systems and applications
- Investigating running processes beyond Task Manager's capabilities
- Identifying process behavior, handles, and loaded modules

## Tools Used
- Process Explorer
- Process Hacker
- TCPView
- TryHackMe – Sysinternals
